# The CrossWild - Admin Panel

A comprehensive admin panel built with React, Vite, Tailwind CSS, and React Router for managing The CrossWild e-commerce website.

![Admin Panel](https://img.shields.io/badge/React-19-blue) ![Vite](https://img.shields.io/badge/Vite-7.3-purple) ![Tailwind](https://img.shields.io/badge/TailwindCSS-3-cyan)

---

## ✨ Features

### 🏠 Dashboard
- **Real-time Statistics**: Total products, blogs, orders, and revenue
- **Sales Analytics**: Interactive charts showing sales trends
- **Order Management**: Quick view of pending and completed orders
- **Recent Orders Table**: Latest customer orders at a glance
- **Product Distribution**: Pie chart showing products by category

### 📦 Products Management
- **CRUD Operations**: Create, Read, Update, Delete products
- **Product Filtering**: Search by name, filter by category
- **Stock Management**: Track inventory levels
- **Image Upload Support**: Product images via URL
- **Category Management**: Organize products into 9 categories
- **Product Attributes**: Sizes, colors, pricing, stock levels
- **Badges**: Mark products as Best Seller, New Arrival, Featured

### 📝 Blog Management
- **Full Blog CRUD**: Complete blog post management
- **Rich Content**: Title, content, featured image
- **Author Information**: Name, designation, profile image
- **Tags System**: Categorize blog posts
- **Publication Date**: Track when posts were published
- **Grid View**: Beautiful card-based layout

### 🛒 Orders Management
- **Order Tracking**: View all customer orders
- **Status Management**: Pending, Processing, Completed, Cancelled
- **Customer Information**: Name, email, order details
- **Order Filtering**: Search and filter by status
- **Real-time Updates**: Instant status changes
- **Order Statistics**: Quick stats for different order statuses

## 🚀 Getting Started

### Installation

1. Navigate to admin folder:
\`\`\`bash
cd admin
\`\`\`

2. Install dependencies:
\`\`\`bash
npm install
\`\`\`

3. Start development server:
\`\`\`bash
npm run dev
\`\`\`

4. Build for production:
\`\`\`bash
npm run build
\`\`\`
